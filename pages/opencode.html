OpenCode

VS Code를 위한 오픈소스 AI 코딩 어시스턴트

⚡ 빠른 시작
  1. VS Code Extensions에서 "OpenCode" 검색
  2. Install 클릭
  3. API 키 설정 (Claude, GPT-4, 또는 Ollama)
  4. Cmd+K (Mac) 또는 Ctrl+K (Windows)로 시작

소개

OpenCode는 VS Code를 위한 오픈소스 AI 코딩 어시스턴트입니다. 다양한 LLM을 지원하며, 인라인 편집, 채팅, 자동완성 기능을 제공합니다.

주요 기능

  • 인라인 편집: 코드 블록 선택 후 즉시 수정
  • 채팅 인터페이스: 사이드바에서 AI와 대화
  • 자동완성: Tab으로 코드 제안 수락
  • 다중 LLM: Claude, GPT-4, Gemini, Ollama 지원
  • 오픈소스: MIT 라이선스, 무료
  • 프라이버시: 로컬 LLM 사용 가능

설치 및 설정

설치

  1. VS Code 열기
  2. Extensions (Ctrl+Shift+X 또는 Cmd+Shift+X)
  3. "OpenCode" 검색
  4. Install 클릭

설정

Claude API 사용

// settings.json
{
  "opencode.provider": "anthropic",
  "opencode.apiKey": "sk-ant-api03-xxx",
  "opencode.model": "claude-sonnet-4-5-20250929"
}

Ollama 사용 (무료)

// settings.json
{
  "opencode.provider": "ollama",
  "opencode.model": "llama3.2",
  "opencode.ollamaUrl": "http://localhost:11434"
}
💡 권장 설정
  • Claude Sonnet 4.5: 최고 품질
  • Ollama Llama 3.2: 무료, 오프라인
  • GPT-4: 범용 성능

사용법

인라인 편집

  1. 코드 블록 선택
  2. Cmd+K (Mac) 또는 Ctrl+K (Windows)
  3. 원하는 변경사항 입력
  4. Enter → AI가 코드 수정

예시:

// 선택한 코드
function add(a, b) {
  return a + b;
}

// Cmd+K 후 입력: "타입스크립트로 변환하고 JSDoc 추가"

// AI가 생성
/**
 * Adds two numbers together
 * @param a - First number
 * @param b - Second number
 * @returns Sum of a and b
 */
function add(a: number, b: number): number {
  return a + b;
}

채팅 인터페이스

  1. 사이드바에서 OpenCode 아이콘 클릭
  2. 채팅 입력창에 질문 또는 요청
  3. AI 응답 확인
  4. 코드 블록 클릭 → 자동 삽입

자동완성

코드를 작성하면 자동으로 제안이 나타납니다. Tab 키로 수락, Esc로 거부.

활용 팁

  • 단축키 익히기: Cmd+K (인라인), Cmd+I (채팅)
  • 컨텍스트 제공: 관련 파일 함께 열어두기
  • 반복 작업: 템플릿 저장하여 재사용
  • Ollama 활용: 간단한 작업은 로컬 LLM
