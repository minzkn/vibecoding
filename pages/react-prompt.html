React 프롬프트 - 기술 스택별 가이드

React 개발을 위한 효과적인 프롬프트 기법을 배웁니다. 컴포넌트, Hooks, 상태管理等 React 특화 프롬프트를 알아봅니다.

개요

React는 컴포넌트 기반의 UI 라이브러리입니다. AI에게 React 코드를 요청할 때에는 React의 특성을 반영한 프롬프트를 작성하면 더 좋은 결과를 얻을 수 있습니다.

📋 React 프롬프트 특징
  • 컴포넌트 구조: 컴포넌트 분리 명시
  • Hooks: useState, useEffect 등 명시
  • 스타일링: CSS Modules, styled-components 등
  • 상태 관리: props, context, 상태 관리 라이브러리
