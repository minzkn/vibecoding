최적화 요청

다음 SQL 쿼리를 최적화해줘.

```sql
SELECT * FROM posts
WHERE created_at > '2024-01-01'
AND author_id IN (SELECT id FROM users WHERE active = true)
ORDER BY created_at DESC
LIMIT 100;
```
