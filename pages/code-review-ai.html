📋 AI 코드 리뷰의 장점

自动化

GitHub Actions로 자동 코드 리뷰를 설정합니다.

GitHub Actions 설정

YAML (.github/workflows/ai-review.yml) 
name: AI Code Review

on:
  pull_request:
    branches: [main]

jobs:
  ai-review:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4
        
      - name: Get diff
        run: |
          git diff HEAD~1 HEAD --unified=5 > diff.txt
          
      - name: AI Review
        env:
          OPENAI_API_KEY: ${{ secrets.OPENAI_API_KEY }}
        run: |
          npx @openai/code-review-cli review diff.txt

      - name: Post Review
        uses: actions/github-script@v7
        with:
          script: |
            github.rest.issues.createComment({
              issue_number: context.issue.number,
              owner: context.repo.owner,
              repo: context.repo.repo,
              body: 'AI 코드 리뷰가 완료되었습니다.'
            })
