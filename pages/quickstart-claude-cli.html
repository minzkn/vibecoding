Claude CLI 5분 시작

Claude CLI를 설치하고 첫 번째 Vibe Coding 프로젝트를 5분 안에 시작하세요.

업데이트 안내: 모델/요금/버전/정책 등 시점에 민감한 정보는 변동될 수 있습니다. 최신 내용은 공식 문서를 확인하세요.
⏱️ 5분 체크리스트
  1. ✅ Claude API 키(Anthropic) 발급 (2분)
  2. ✅ Claude CLI 실행 (1분)
  3. ✅ 첫 번째 프로젝트 생성 (2분)

필요한 것: 터미널, Node.js (14+), 신용카드 (무료 크레딧 제공)

Claude CLI란?

Claude CLI (Claude Code)는 Anthropic이 공식 제공하는 커맨드라인 인터페이스입니다. 터미널에서 Claude와 대화하며 코드를 생성하고, 파일을 수정하며, 프로젝트를 관리할 수 있습니다.

왜 Claude CLI인가?

  • 공식 도구: Anthropic이 직접 개발 및 유지보수
  • 최신 모델: Claude 4.6 Opus, 4.5 Sonnet 즉시 지원
  • 최고 품질: 코드 생성 품질이 다른 도구들보다 우수
  • 간단한 사용: npx claude 한 줄로 시작
  • MCP 지원: Model Context Protocol 네이티브 통합
⚠️ API 키 필요

Claude CLI는 Anthropic API를 사용하므로 API 키가 필요합니다. 무료 크레딧(변동)이 제공되지만, 이후 유료입니다.

무료로 시작하고 싶다면: Ollama 로컬 시작을 참고하세요.

🧭 Codex 대비 안내

Codex는 OpenAI 계정/키로 시작합니다. CLI는 로그인 흐름 또는 API 키 인증을 제공합니다.

Codex 앱은 macOS 환경에서 제공됩니다. Codex 가이드 참고.

Step 1: Claude API 키(Anthropic) 발급

Claude CLI를 사용하려면 먼저 Anthropic Console에서 API 키를 발급받아야 합니다.

1-1. 계정 생성

  1. https://console.anthropic.com 접속
  2. Sign Up 클릭 (Google 계정 또는 이메일)
  3. 이메일 인증 완료
🎁 무료 크레딧

신규 가입 시 변동 크레딧이 제공됩니다. 이는 약 1,000,000 입력 토큰 (A4 용지 약 3,000장)에 해당합니다.

초보자가 연습하기에 충분한 양입니다!

1-2. API 키 생성

  1. 로그인 후 좌측 메뉴에서 API Keys 클릭
  2. Create Key 버튼 클릭
  3. 키 이름 입력 (예: "my-claude-cli")
  4. Create Key 확인
  5. 생성된 키를 복사 (⚠️ 한 번만 표시됩니다!)
# API 키 예시 (실제로는 훨씬 길어요)
sk-ant-api03-xxxxxxxxxxxxxxxxxxxxxxxxxxx...
🔐 보안 주의사항
  • API 키는 절대 공개하지 마세요 (GitHub, 블로그 등)
  • 키를 분실하면 재생성 필요
  • 환경 변수로 관리 권장

API 키 안전 관리 가이드 →

1-3. 환경 변수 설정

API 키를 환경 변수로 설정하면 매번 입력할 필요가 없습니다.

macOS / Linux (bash/zsh)

# ~/.bashrc 또는 ~/.zshrc에 추가
export ANTHROPIC_API_KEY="sk-ant-api03-xxxxxxx"

# 즉시 적용
source ~/.bashrc  # 또는 source ~/.zshrc

Windows (PowerShell)

# 현재 세션만
$env:ANTHROPIC_API_KEY = "sk-ant-api03-xxxxxxx"

# 영구 설정 (시스템 환경 변수)
[System.Environment]::SetEnvironmentVariable("ANTHROPIC_API_KEY", "sk-ant-api03-xxxxxxx", "User")

Windows (CMD)

# 현재 세션만
set ANTHROPIC_API_KEY=sk-ant-api03-xxxxxxx

1-4. API 키 확인

# 설정 확인 (macOS/Linux)
echo $ANTHROPIC_API_KEY

# Windows (PowerShell)
echo $env:ANTHROPIC_API_KEY

키가 출력되면 성공입니다! ✅

Step 2: Claude CLI 실행

Claude CLI는 설치 없이 npx로 바로 실행할 수 있습니다.

2-1. 사전 요구사항 확인

Node.js 14 이상이 필요합니다: 

# Node.js 버전 확인
node --version

# 예상 출력
v18.17.0  # 14 이상이면 OK
Node.js가 없다면?

Node.js 공식 사이트에서 LTS 버전을 다운로드하세요.

또는:

  • macOS: brew install node
  • Windows: winget install OpenJS.NodeJS
  • Linux: sudo apt install nodejs npm

2-2. Claude CLI 실행

# npx로 즉시 실행 (설치 불필요)
npx claude

# 처음 실행 시 패키지 다운로드 메시지가 나올 수 있습니다
Need to install the following packages:
  @anthropic-ai/claude-cli
Ok to proceed? (y)

# y 입력 후 Enter

잠시 후 Claude CLI가 시작됩니다: 

╭────────────────────────────────────────╮
│  Welcome to Claude Code CLI            │
│  Type your request or 'help' for info │
╰────────────────────────────────────────╯

💬 You: _
💡 전역 설치 (선택사항)

자주 사용한다면 전역 설치가 편리합니다: 

# 전역 설치
npm install -g @anthropic-ai/claude-cli

# 이제 짧은 명령어로 실행
claude

Claude와 대화하며 기능을 추가해봅시다: 

💬 You: "좋아! 이제 다음 기능들 추가해줘:
1. 다크모드 토글 버튼
2. 완료된 항목을 아래로 이동
3. '모두 삭제' 버튼

🤖 Claude: "모든 기능을 추가했습니다!

✨ 변경사항:
1. index.html - 다크모드 토글, 모두 삭제 버튼, 우선순위 select 추가
2. style.css - 다크모드 스타일, 우선순위별 색상 (빨강/노랑/초록)
3. script.js - 자동 정렬 로직, 다크모드 전환, 일괄 삭제 기능

localStorage에 다크모드 설정도 저장됩니다."

페이지를 새로고침하면 새로운 기능들이 적용됩니다!

3-6. 코드 이해하기

생성된 코드가 이해되지 않는다면 Claude에게 물어보세요: 

💬 You: "script.js에서 localStorage.setItem은 어떻게 작동해?"

🤖 Claude: "localStorage.setItem은 브라우저의 로컬 스토리지에 데이터를 저장하는 함수입니다.

문법:
```javascript
localStorage.setItem(key, value)
```

우리 앱에서는:
```javascript
localStorage.setItem('todos', JSON.stringify(todosArray));
```

- 'todos': 저장할 데이터의 키 (이름)
- JSON.stringify(): JavaScript 객체를 문자열로 변환
  (localStorage는 문자열만 저장 가능)

데이터 읽기:
```javascript
const saved = localStorage.getItem('todos');
const todos = JSON.parse(saved); // 문자열을 객체로 변환
```

이렇게 하면 브라우저를 닫았다 열어도 데이터가 유지됩니다!"

문제 해결

일반적인 문제

❌ "API key not found" 오류

원인: 환경 변수가 설정되지 않음

해결:

# 환경 변수 확인
echo $ANTHROPIC_API_KEY

# 비어있다면 다시 설정
export ANTHROPIC_API_KEY="sk-ant-api03-xxxxxxx"

❌ "npx: command not found"

원인: Node.js가 설치되지 않음

해결: Node.js 설치

❌ "Rate limit exceeded"

원인: 너무 많은 요청을 빠르게 보냄

해결: 잠시 기다렸다가 재시도 (보통 1분)

❌ 파일이 생성되지 않음

원인: 디렉토리 권한 문제 또는 요청이 불명확

해결:

  • 현재 디렉토리 쓰기 권한 확인
  • 더 구체적인 요청으로 다시 시도
  • 파일 경로를 명시적으로 지정

❌ "Insufficient credits"

원인: 무료 크레딧 소진

해결:

  • Billing 페이지에서 결제 수단 추가
  • 또는 Ollama로 전환 (무료)
