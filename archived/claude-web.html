Claude.ai 웹 인터페이스

Claude.ai는 브라우저에서 바로 사용할 수 있는 웹 인터페이스입니다. 코드 없이 Claude의 모든 기능을 활용하고, Projects와 Artifacts로 생산성을 극대화하세요.

시작하기

접속 방법

  1. claude.ai 접속
  2. 이메일 또는 Google 계정으로 로그인
  3. 무료 또는 Pro 플랜 선택
💡 Free vs Pro

Free: 제한된 사용량, Sonnet 모델
Pro ($20/월): 5배 많은 사용량, Opus/Sonnet/Haiku 모델 선택, Priority access

채팅 인터페이스

기본 대화

입력창에 질문이나 요청을 입력하고 Enter를 누르면 Claude가 응답합니다. Shift+Enter로 줄바꿈이 가능합니다.

파일 업로드

입력창 옆의 📎 아이콘을 클릭하거나 드래그 앤 드롭으로 파일을 업로드할 수 있습니다.

지원 파일 형식:

  • 이미지: PNG, JPEG, GIF, WebP (최대 5개, 각 5MB)
  • 문서: PDF, TXT, CSV, DOCX, XLSX (최대 10MB)
  • 코드: .py, .js, .java, .cpp 등 모든 텍스트 파일
💡 프로 팁: 다중 파일 업로드

여러 파일을 한 번에 업로드하여 비교 분석, 통합 요약 등이 가능합니다. 예: "이 3개 계약서를 비교하고 차이점을 알려주세요"

Projects 기능

Projects는 관련 대화와 지식을 한 곳에 모아 관리하는 기능입니다 (Pro 전용).

프로젝트 생성

  1. 좌측 사이드바에서 "Projects" 클릭
  2. "New Project" 버튼 클릭
  3. 프로젝트 이름 및 설명 입력
  4. Custom Instructions 작성 (선택)

Custom Instructions

프로젝트별로 Claude의 역할, 규칙, 출력 형식을 정의할 수 있습니다. 일종의 시스템 프롬프트로, 해당 프로젝트의 모든 대화에 자동 적용됩니다.

Custom Instructions 예시 
당신은 Python 백엔드 개발 전문가입니다.

역할:
- Django/FastAPI 코드 리뷰
- API 설계 자문
- 성능 최적화 제안

규칙:
- 모든 코드에 타입 힌트 포함
- PEP 8 스타일 준수
- 보안 취약점 항상 점검

출력 형식:
- 코드는 ```python 블록 사용
- 설명은 간결하게 (3줄 이내)

프로젝트 지식 (Knowledge)

프로젝트에 문서, 코드, 데이터를 업로드하면 Claude가 자동으로 참조합니다. 최대 100개 파일, 총 30MB까지 업로드 가능합니다.

활용 예시:

  • 회사 스타일 가이드 업로드 → 일관된 콘텐츠 생성
  • API 문서 업로드 → 정확한 코드 예제 생성
  • 과거 프로젝트 코드 업로드 → 유사 패턴 재사용

Artifacts

Artifacts는 Claude가 생성한 콘텐츠를 별도 패널에 표시하고 편집할 수 있는 기능입니다.

지원 타입

타입 설명 활용
코드 Python, JavaScript, HTML 등 즉시 복사, 다운로드
문서 Markdown, 텍스트 편집 후 Export
SVG 벡터 그래픽 다이어그램, 아이콘 생성
React 컴포넌트 실행 가능한 UI 실시간 미리보기

Artifact 생성 프롬프트

Artifact 생성 예시 
# 코드 Artifact
"Python으로 피보나치 수열 생성 함수를 작성하고 Artifact로 보여주세요"

# React Artifact
"할 일 목록 앱을 React로 만들어주세요 (Artifact로)"

# SVG Artifact
"플로우차트를 SVG로 그려주세요: 사용자 로그인 → 인증 → 대시보드"

# 문서 Artifact
"제품 출시 보도자료를 작성하고 Artifact로 표시해주세요"

Artifact 편집

Artifact 내에서 직접 편집할 수 있으며, "Remix this Artifact"로 Claude에게 수정을 요청할 수 있습니다.

💡 프로 팁: 반복 개선

Artifact를 생성한 후 "이 부분을 수정해주세요", "여기에 기능 추가"처럼 구체적으로 요청하면 Claude가 기존 Artifact를 업데이트합니다. 코드 개발 시 매우 유용합니다.

고급 기능

모델 선택 (Pro)

Pro 플랜에서는 대화마다 다른 모델을 선택할 수 있습니다:

  • Opus: 복잡한 분석, 창작 작업
  • Sonnet: 일반적인 대화, 코딩 (균형)
  • Haiku: 빠른 응답이 필요한 간단한 작업

대화 관리

  • 대화 이름 변경: 대화 제목 클릭
  • 대화 검색: 좌측 검색창으로 과거 대화 찾기
  • 대화 고정: 중요한 대화를 상단 고정
  • 대화 삭제: 대화 옆 ⋮ 메뉴 → Delete

키보드 단축키

단축키 기능
Cmd/Ctrl + K 새 대화 시작
Cmd/Ctrl + Shift + L 최근 대화로 이동
Cmd/Ctrl + / 단축키 목록 보기
Shift + Enter 줄바꿈 (메시지 전송 안 함)

활용 팁

1. 프로젝트로 컨텍스트 관리

같은 주제의 대화는 하나의 프로젝트에 모아두세요. Custom Instructions와 Knowledge를 활용하면 매번 배경 설명을 반복할 필요가 없습니다.

2. 대화 이어가기

같은 대화 스레드에서 후속 질문을 하면 Claude가 이전 컨텍스트를 기억합니다. 새 대화를 시작하면 컨텍스트가 사라지므로 주의하세요.

3. 명확한 지시

웹 인터페이스에서도 프롬프트 엔지니어링 원칙이 적용됩니다:

  • 구체적인 요청
  • 예시 제공
  • 출력 형식 지정
  • 단계별 지시

4. 파일 최적화

긴 PDF는 필요한 페이지만 추출하여 업로드하세요. 토큰 사용량을 줄이고 응답 속도가 빨라집니다.

5. 대화 내보내기

중요한 대화는 내보내기(Export) 기능으로 백업하세요. 대화 우측 상단 ⋮ → Export conversation.

실전 활용 사례

사례 1: 코드 리뷰 프로젝트

프로젝트 설정 
프로젝트명: "Python 코드 리뷰"

Custom Instructions:
- Python 코드 리뷰 전문가
- PEP 8, 보안, 성능 중점 검토
- 개선 코드 제시

Knowledge:
- 회사 코딩 스타일 가이드.pdf
- 과거 리뷰 예시.md

사례 2: 콘텐츠 제작 워크플로우

단계별 Artifact 활용 
1단계: "블로그 주제 10개 브레인스토밍" → 리스트 Artifact

2단계: "3번 주제로 아웃라인 작성" → 문서 Artifact

3단계: "아웃라인 기반 전체 글 작성" → 문서 Artifact (편집 가능)

4단계: "이 글을 SNS용 요약본 3개로 만들어주세요" → 리스트 Artifact

사례 3: 데이터 분석

CSV 파일 업로드 → "이 데이터를 분석하고 Python pandas 코드 생성" → Artifact로 코드 받기 → 로컬에서 실행

제한사항

사용량 제한

  • Free: 제한된 메시지 수 (정확한 수는 비공개, 수요에 따라 변동)
  • Pro: 훨씬 많은 메시지, 하지만 무제한은 아님

파일 크기 제한

  • 이미지: 개당 5MB
  • 문서: 개당 10MB
  • 프로젝트 Knowledge: 총 30MB

실시간 정보 없음

웹 브라우저에서 사용하지만, Claude는 인터넷에 접근할 수 없습니다. 2025년 1월 이후 정보는 모릅니다 (Knowledge Cutoff).
