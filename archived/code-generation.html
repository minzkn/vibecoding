코드 생성

Claude는 강력한 코딩 어시스턴트로, 다양한 프로그래밍 언어의 코드를 생성하고, 리팩토링하며, 버그를 찾아내는 데 탁월한 능력을 발휘합니다.

개요

Claude를 코드 생성에 활용하면 개발 속도를 크게 향상시킬 수 있습니다. 단순한 코드 작성뿐만 아니라 테스트 코드, 문서화, 최적화까지 지원합니다.

💡 주요 활용 분야
  • 함수/클래스 생성: 요구사항을 코드로 변환
  • 알고리즘 구현: 복잡한 로직 작성
  • 리팩토링: 코드 품질 개선
  • 디버깅: 버그 찾기 및 수정
  • 테스트 작성: 단위/통합 테스트 생성
  • 문서화: 코드 주석 및 README 작성

Vibe Coding으로 코드 생성

Vibe Coding은 자연어 대화만으로 코드를 생성하는 혁신적인 개발 방식입니다. Claude Code CLI와 API 모두에서 Vibe Coding 패러다임을 활용할 수 있습니다.

🎯 Vibe Coding의 핵심 원칙
  • 의도 중심: "무엇을 만들고 싶은지"만 설명
  • 구문 무관: 언어 문법을 몰라도 가능
  • 빠른 반복: 대화로 즉시 수정
  • 다양한 언어: 동일한 의도를 여러 언어로 생성

def remove_duplicates_and_sort(items: list) -> list:
    """
    리스트에서 중복을 제거하고 정렬된 결과를 반환합니다.

    Args:
        items: 입력 리스트 (모든 타입의 요소 가능)

    Returns:
        중복이 제거되고 정렬된 리스트

    Examples:
        >>> remove_duplicates_and_sort([3, 1, 2, 1, 3])
        [1, 2, 3]
        >>> remove_duplicates_and_sort(['c', 'a', 'b', 'a'])
        ['a', 'b', 'c']
    """
    return sorted(set(items))

클래스 설계

프롬프트 
TypeScript로 간단한 Stack 자료구조를 구현해주세요.

요구사항:
- Generic 타입 지원
- push, pop, peek, isEmpty, size 메서드
- 각 메서드에 JSDoc 주석
- 오류 처리 (빈 스택에서 pop/peek 시 에러)
TypeScript (Claude 응답) 
/**
 * Generic Stack 자료구조 구현
 */
class Stack<T> {
  private items: T[] = [];

  /**
   * 스택에 요소를 추가합니다.
   * @param item - 추가할 요소
   */
  push(item: T): void {
    this.items.push(item);
  }

  /**
   * 스택에서 최상위 요소를 제거하고 반환합니다.
   * @returns 제거된 요소
   * @throws {Error} 스택이 비어있을 경우
   */
  pop(): T {
    if (this.isEmpty()) {
      throw new Error("Stack is empty");
    }
    return this.items.pop()!;
  }

  /**
   * 스택의 최상위 요소를 반환합니다 (제거하지 않음).
   * @returns 최상위 요소
   * @throws {Error} 스택이 비어있을 경우
   */
  peek(): T {
    if (this.isEmpty()) {
      throw new Error("Stack is empty");
    }
    return this.items[this.items.length - 1];
  }

  /**
   * 스택이 비어있는지 확인합니다.
   * @returns 비어있으면 true, 아니면 false
   */
  isEmpty(): boolean {
    return this.items.length === 0;
  }

  /**
   * 스택의 크기를 반환합니다.
   * @returns 스택에 있는 요소의 개수
   */
  size(): number {
    return this.items.length;
  }
}

from typing import List

# 상수 정의
THRESHOLD = 100
HIGH_RATE = 0.1
LOW_RATE = 0.05

def calculate_discount(value: float) -> float:
    """
    개별 값에 대한 할인을 계산합니다.

    임계값(100)보다 큰 값은 10% 할인,
    그 외는 5% 할인이 적용됩니다.

    Args:
        value: 할인을 계산할 값

    Returns:
        할인이 적용된 값
    """
    rate = HIGH_RATE if value > THRESHOLD else LOW_RATE
    return value * rate

def calculate_total_discount(data: List[float]) -> float:
    """
    데이터 리스트의 총 할인 금액을 계산합니다.

    Args:
        data: 할인을 계산할 값들의 리스트

    Returns:
        총 할인 금액
    """
    return sum(calculate_discount(value) for value in data)

다음 코드에 버그가 있습니다. 찾아서 수정해주세요:

```javascript
function findMax(arr) {
  let max = 0;
  for (let i = 0; i < arr.length; i++) {
    if (arr[i] > max) {
      max = arr[i];
    }
  }
  return max;
}
```

테스트 케이스:
- findMax([1, 5, 3, 9, 2]) → 9 (정상)
- findMax([-5, -2, -8, -1]) → -1 (실패, 0이 반환됨)
JavaScript (수정된 코드) 
/**
 * 배열에서 최댓값을 찾습니다.
 * @param {number[]} arr - 숫자 배열
 * @returns {number} 배열의 최댓값
 * @throws {Error} 빈 배열인 경우
 */
function findMax(arr) {
  // 버그 수정 1: 빈 배열 검사
  if (arr.length === 0) {
    throw new Error("Array cannot be empty");
  }

  // 버그 수정 2: max를 첫 번째 요소로 초기화
  // (음수만 있는 배열에서 0보다 작은 값을 처리하기 위함)
  let max = arr[0];

  for (let i = 1; i < arr.length; i++) {
    if (arr[i] > max) {
      max = arr[i];
    }
  }

  return max;
}

// 또는 더 간결한 방법:
function findMax(arr) {
  if (arr.length === 0) {
    throw new Error("Array cannot be empty");
  }
  return Math.max(...arr);
}

```typescript
function isPalindrome(str: string): boolean {
  const cleaned = str.toLowerCase().replace(/[^a-z0-9]/g, '');
  return cleaned === cleaned.split('').reverse().join('');
}
```

TypeScript (테스트 코드) 
import { isPalindrome } from './palindrome';

describe('isPalindrome', () => {
  describe('기본 케이스', () => {
    it('단순 회문을 감지해야 함', () => {
      expect(isPalindrome('racecar')).toBe(true);
      expect(isPalindrome('level')).toBe(true);
    });

    it('회문이 아닌 경우 false 반환', () => {
      expect(isPalindrome('hello')).toBe(false);
      expect(isPalindrome('world')).toBe(false);
    });
  });

  describe('대소문자 처리', () => {
    it('대소문자를 무시해야 함', () => {
      expect(isPalindrome('RaceCar')).toBe(true);
      expect(isPalindrome('LeVeL')).toBe(true);
    });
  });

  describe('특수 문자 및 공백', () => {
    it('공백과 특수 문자를 무시해야 함', () => {
      expect(isPalindrome('A man, a plan, a canal: Panama')).toBe(true);
      expect(isPalindrome('race a car')).toBe(false);
    });
  });

  describe('엣지 케이스', () => {
    it('빈 문자열은 회문으로 처리', () => {
      expect(isPalindrome('')).toBe(true);
    });

    it('단일 문자는 회문', () => {
      expect(isPalindrome('a')).toBe(true);
    });

    it('숫자가 포함된 회문', () => {
      expect(isPalindrome('1a2b2a1')).toBe(true);
    });
